Переделай отображение товаров на мобильной версии сайта (< 768px). Две части: главная страница и каталог.

=== ЧАСТЬ 1: КАТАЛОГ (/catalog) — мобильная версия ===

Сейчас карточки товаров на мобильном занимают всю ширину экрана (1 карточка в ряд). Нужно сделать компактную сетку 2 колонки.

Компактная карточка содержит ТОЛЬКО:
- Фото товара (квадратное, aspect-ratio 1:1, object-fit cover)
- Бренд (мелким текстом, розовый, uppercase)
- Название товара (1 строка, overflow hidden, text-overflow ellipsis)
- Цена
- Кнопка «Купить» — розовая, маленькая, под ценой. Вся карточка и кнопка ведут на /product/:id
- Если товар продан: лента «ПРОДАНО» поверх фото (как сейчас), цена перечёркнутая, кнопка «Купить» заменяется на неактивную «Продано»

Убрать с компактной карточки (всё это есть на странице товара):
- Размер
- «Таблица размеров» (аккордеон)
- Кнопку «Написать в Telegram»
- Кнопку «Купить на Авито»
- Ироничную подпись (caption)
- Замеры штанов

CSS: grid с grid-template-columns: repeat(2, 1fr), gap: 8px. Карточки с border-radius: 10px, белый фон, небольшой padding (6-8px).

На десктопе (>= 768px) оставить текущий вид карточек как есть, но можно сделать 4 в ряд вместо 3.

=== ЧАСТЬ 2: ГЛАВНАЯ СТРАНИЦА (/) — секции по категориям ===

Заменить текущий блок «Каталог» (3 избранных товара + кнопка «Смотреть весь ассортимент») на секции по категориям. Показывать секции только на мобильном (< 768px). На десктопе оставить как есть.

Категории (в таком порядке): Обувь (shoes), Верх (tops), Низ (bottoms), Аксессуары (accessories).

Каждая секция:
- Заголовок категории слева, справа — ссылка «Смотреть все →» ведёт на /catalog?category=[категория]
- Горизонтально прокручиваемый ряд компактных карточек (те же компактные карточки, что в каталоге)
- Показывать до 8 товаров на секцию (только доступные, status=available, сортировка по дате — новые первые)
- В конце ряда — карточка-кнопка «→» того же размера, ведёт на /catalog?category=[категория]
- Горизонтальный скролл: overflow-x: auto, scrollbar скрыт (-webkit-scrollbar: none), scroll-snap-type: x mandatory
- Каждая карточка: min-width: 140px, max-width: 150px
- Если в категории меньше 2 товаров — секцию не показывать

Данные загружать через существующий API (GET /api/products), группировать по category на фронтенде.

=== ОБЩЕЕ ===
- Все карточки кликабельные целиком (обёрнуты в <Link to="/product/:id"> или onClick с navigate)
- Стилизация в розовой гамме сайта
- Плавные переходы, border-radius: 10px
- На десктопе ничего не ломать — изменения только для мобильной версии
