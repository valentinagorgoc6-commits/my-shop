Двойная тема сайта. Этап 3 — splash screen и переключатели в хедере.
Этап 2 уже сделан: CSS-переменные работают, ThemeContext создан, цвета заменены. Теперь нужно дать пользователю возможность выбирать тему.
1. SPLASH SCREEN — при первом посещении сайта (нет pickme-gender в localStorage):

Показать полноэкранный экран поверх всего контента (z-index: 1000, position: fixed, inset: 0)
Фон: мягный градиент или нейтральный светлый цвет
По центру:

Логотип «PickMe Store» (шрифт Cormorant Garamond)
Подпись: «Выбери свой стиль»
Две карточки рядом:

Для неё — розовый акцент (#f04586), иконка или эмодзи (например 👠 или что-то стильное), подпись «Женская коллекция»
Для него — синий акцент (#0074c4), иконка или эмодзи (например 👟), подпись «Мужская коллекция»


Карточки с hover-эффектом (подъём + тень + усиление цвета)


При клике:

Записать выбор в localStorage (pickme-gender)
Установить тему через ThemeContext (setGender)
Splash screen плавно исчезает (opacity 0, затем display: none или unmount)
Анимация появления — fade in при открытии, fade out при выборе (0.4s ease)


Splash screen НЕ показывается повторно если в localStorage уже есть выбор
Splash screen НЕ показывается на страницах /product/:id и /gift/:id — только на главной

2. ХЕДЕР — женская версия (gender === 'female'):
Текущий хедер остаётся как есть, но добавь одну кнопку:

Кнопка «Для него» — маленькая, стиль: синий текст #0074c4, фон rgba(0,100,190,0.06), бордер rgba(0,100,190,0.12), border-radius: 8px
При клике: setGender('male') — сайт переключается на мужскую тему
Кнопку расположить в правой части хедера, перед кнопкой «Написать мне» (или «Telegram»)
Никакой кнопки светлая/тёмная в женской версии НЕТ

3. ХЕДЕР — мужская версия (gender === 'male'):
Хедер мужской версии должен содержать (справа налево):

Кнопка «Написать мне» — стиль: акцентный цвет (var(--color-primary)), аналог текущей кнопки
Кнопка переключения темы (светлая/тёмная):

Иконка луны (☽) + текст «Тёмная» — когда сейчас светлая тема
Иконка солнца (☀) + текст «Светлая» — когда сейчас тёмная тема
Стиль: нейтральный фон (var(--color-surface-alt)), бордер (var(--color-border)), border-radius: 8px
При клике: toggleMode() — переключает между male и male-dark


Кнопка «Для неё» — розовый текст, стиль аналогичный: фон rgba(240,69,134,0.06), бордер rgba(240,69,134,0.12)

При клике: setGender('female') — переключает на женскую тему



Порядок кнопок в хедере: навигация ... | «Для неё» | «☽ Тёмная» | «Написать мне»
4. НАВИГАЦИЯ мужской версии:
Заменить текущие ссылки навигации на:

Каталог
Обо мне
Отзывы
FAQ

Шрифт навигации: Montserrat, 12px, цвет var(--color-text-muted), hover — var(--color-primary)
5. ЛОГОТИП мужской версии:

Текст: «PickMe Store» шрифтом Cormorant Garamond
«PickMe» — цвет var(--color-primary), italic, font-weight: 600
«Store» — цвет var(--color-text-heading), font-weight: 500
Под логотипом мелкая подпись: «ПикМи — магазин брендовых товаров» — font-size: 10px, цвет var(--color-text-muted)

6. ПЛАВНЫЕ ПЕРЕХОДЫ:

При переключении gender или mode — все элементы страницы плавно меняют цвета (transition 0.3s ease уже должен быть из этапа 2)
URL НЕ меняется при смене темы
Скролл страницы сохраняется при переключении

7. МОБИЛЬНАЯ ВЕРСИЯ (< 768px):

Splash screen: карточки «Для неё / Для него» расположить вертикально (flex-direction: column), чуть меньше размер
Хедер: кнопки «Для неё/него» и «Тёмная/Светлая» — показывать только иконки без текста, уменьшить padding
Навигация на мобильном скрыта (как сейчас)

ВАЖНО:

После выбора на splash screen пользователь сразу видит главную страницу в выбранной теме
Контент страницы пока одинаковый для обоих полов — фильтрация каталога по gender будет в этапе 4
Шрифты Cormorant Garamond и Montserrat уже подключены в этапе 2
Не ломай существующую розовую тему — она должна выглядеть точно так же как до всех изменений
